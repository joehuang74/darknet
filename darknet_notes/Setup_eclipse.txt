1. cd ~/git
   git clone https://github.com/pjreddie/darknet.git darknet.git

2. Eclipse
   File -> New -> Makefile project with existing codes -> Linux GCC

3. Modify Makefile to add debug flags for build
   CFLAGS += -DDEBUG -g3

4. Project -> Build All

5. Right click on the compiled executable "darknet" 
   -> Debug as -> Debug Configuration -> set arguments for running

6. Sources: src, examples/darknet.c
